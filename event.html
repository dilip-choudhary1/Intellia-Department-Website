Classify faces Using an Unsupervised approach

Face recognition and face clustering are different. When performing face the recognition we are applying supervised learning where we have both examples of images of faces we want to recognize along with the names that correspond to each face (i.e., the “class labels”). But in face clustering we need to perform unsupervised learning — we have only the faces themselves with no names/labels. From there we need to identify and count the number of unique people in a dataset.
